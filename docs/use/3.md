# 快速开始

安装完成后，您可以直接在聊天中发送消息，OIer-Helper 会自动识别您的意图并响应。

## 典型对话示例

### 1. 分析题目
- 您发送：  
  `https://www.luogu.com.cn/problem/P3371`  
  或  
  `帮我分析一下洛谷 P3371 这题`
- 助手会返回：题目判断、解法思路、关键观察、算法步骤、正确性说明、复杂度分析、C++17 参考代码、易错点。

### 2. 讲解算法
- 您发送：  
  `讲一下 Dijkstra 算法`
- 助手会返回：适用场景、核心思想、算法流程、正确性证明、复杂度、模板代码、常见变形、易错点、练习题推荐。

### 3. 代码调试
- 您发送：  
  `这道题 https://codeforces.com/contest/1900/problem/A 我的代码如下（粘贴代码），为什么 WA？`
- 助手会先分析题目正确做法，然后逐项对比您的代码，指出错误位置、原因和修改建议，并给出修正版代码。

### 4. 推荐题目
- 您发送：  
  `推荐几道线段树的入门题`  
  或  
  `有没有洛谷上的树形 DP 题目？`
- 助手会返回题目列表（题号、标题、难度、链接），并给出简要说明和练习建议。**默认不给题解**，您可以随后要求分析其中某道。

## 常用提问方式

- **直接发链接**：自动进入题目分析模式。
- **“讲一下 + 算法名”**：进入算法讲解模式。
- **“推荐 + 算法/难度”**：进入题目推荐模式。
- **“题目链接 + 我的代码”**：进入调试模式。

## 提示

- 如果助手未按预期响应，请尝试更明确的表述（例如“请帮我分析这道题”）。
- 对于代码调试，请提供完整的代码和错误信息（WA / TLE / RE）。
