# 安装方法

## 环境要求

- 一个支持 Skill 的 AI 工具（如 **WorkBuddy**）。
- 能够访问互联网（部分功能依赖联网搜索）。

## 安装步骤

1. 前往 [下载](/install/) 页面，选择版本和安装方法并按提示下载/安装 Skill；

2. **如果是从 SkillHub 提示词安装可以跳过本步骤。**
   打开 AI 工具对应的 Skill 文件夹，将 Skill 的 ZIP 压缩包放入其中；

3. 安装完成后，您可以在对话中直接调用 OIer-Helper。

> **注意**：本 Skill 需要联网权限才能搜索题解、算法资料和推荐题目。请确保您的 AI 工具允许 Skill 访问网络。
