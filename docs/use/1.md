# 项目介绍

## OIer-Helper 是什么？

**OIer-Helper** 是一个面向 OI / ICPC / 算法竞赛学习者的智能助手 Skill，可集成到 WorkBuddy 等支持 Skill 的 AI 工具中。它以 Claude 等大语言模型为后端，通过精细化的提示词工程，为竞赛选手提供专业、准确、负责任的学习帮助。

## 适用人群

- 信息学奥林匹克竞赛（NOI / NOIP / CSP）参赛者
- ICPC / ACM 竞赛选手
- 算法爱好者与刷题者
- 备考算法面试的程序员

## 核心特点

- **多任务支持**：题目分析、算法讲解、代码调试、题目推荐，一站式满足学习需求。
- **多平台兼容**：覆盖国内外主流 OJ（洛谷、Codeforces、AtCoder、AcWing、LibreOJ、POJ 等）。
- **注重理解**：不只给代码，强调思路、证明与复杂度，帮助你举一反三。
- **尊重学术规范**：总结公开题解，不搬运完整代码，不绕过付费内容，并提醒用户遵守社区规则。
- **默认中文 + C++17**：降低交流门槛，代码风格统一。

## 设计原则

- 准确性优先：不编造来源，不确定时明确说明。
- 诚实透明：未找到题解时如实告知，代码未经运行验证时加以提醒。
- 用户友好：语气耐心，鼓励式纠错，避免专业术语堆砌。
