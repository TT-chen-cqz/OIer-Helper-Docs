# 常见问题

## 1. 为什么我发了一个链接，助手没有分析，而是回了其他内容？
可能是链接格式不被识别，或者助手误判为其他任务。请尝试明确说“请分析这道题”，或直接复制题面。

## 2. 助手说我“没有找到可靠题解”，我该怎么办？
此时助手会基于题意推导参考解法，但会提醒代码未经验证。您可以在网上再搜一下题解，或者用小数据自行测试代码是否正确。

## 3. 可以指定使用 Python 或 Java 吗？
目前默认只提供 C++17 代码。如果您需要其他语言，可以提出，但助手优先支持 C++17，且部分算法模板可能缺少其他语言版本。

## 4. 推荐题目时为什么没有给题解？
这是有意设计的。我们希望您先自主思考，选定题目后再单独请求分析，这样学习效果更好。如果您需要题解，可以在推荐后说“请分析第 3 题”。

## 5. 代码调试时，助手给出的修正版代码一定正确吗？
不一定。助手会尽力基于标准思路修改，但由于无法运行测试，仍可能存在疏漏。请务必自行测试边界数据和样例。

## 6. 我可以要求助手给出多组数据的代码吗？
可以。如果题目输入包含多组测试数据，助手会在代码中正确处理（使用 `while (cin >> n)` 等），并提醒多测清空。

## 7. 助手会爬取需要登录的 OJ 吗？
不会。助手只访问公开页面和公开题解，不绕过任何付费或登录限制。

## 8. 为什么有时候回答很慢？
因为需要联网搜索和整理信息，网络延迟会影响速度。请耐心等待。

## 9. 如何反馈问题或建议？
您可以在项目 [GitHub 仓库](https://github.com/TT-chen-cqz/OIer-Helper) 提交 Issue 或 Pull Request。

## 10. 这个 Skill 会收费吗？
目前 OIer-Helper 是完全免费开源的（GPL v3），无任何收费计划。
