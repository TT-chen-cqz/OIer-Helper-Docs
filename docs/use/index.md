# OIer-Helper 快速上手

欢迎使用 **OIer-Helper** —— 专为算法竞赛学习者打造的智能助手 Skill。

本 Skill 可帮助你：
- 分析任意主流 OJ 题目，给出 C++17 参考代码与详细题解；
- 系统学习算法与数据结构，理解核心思想、证明与复杂度；
- 定位代码中的 Bug，提供修正建议；
- 按算法、难度、平台搜索推荐练习题目。
